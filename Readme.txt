different types of commands executed in the terminal
$ mkdir story (and cd also)
$ls (files and the folder in a folder)
$ls -a (ls with the result having hidden files)
$ git init (initializing git)
$ git status (status of whether the file is in the git staging area)
$ git add filename (adds the file into git staging area)
$ git add . (for adding every filename)
$ git commit -m "message about the files in present tense for example ~complete file1" (commiting/ putting it to the LOCAL git repository)
$ git log (tells the time date of a file last commited into the LOCAL git repository)
$ git diff filename (to show the difference between the latest update and a previous version of file)
$ git checkout filename (revert back or rollback a file to a previous version)
